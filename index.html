j4status-plugins

Various plugins to enhance your j4status experience

j4status-plugins adds several plugins to the official set shipped with j4status.

j4status-plugins is a part of the j4tools tools set.

Dependencies

  • GLib
  • xsltproc and DocBook man page XSL stylesheets (for man pages generation)
  • For the ALSA input plugin:
    • alsa-lib
  • For the i3 focus input plugin:
    • i3ipc-glib
  • For the NetworkManager input plugin:
    • libnm-util (0.9 or later)
    • libnm-glib (0.9 or later)

Git repository

The repository is hosted on GitHub: j4status-plugins Git repository

You can clone the project with Git by running:
$ git clone https://github.com/j4tools/j4status-plugins
